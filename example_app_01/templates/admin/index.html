Desktop Database Application

This is a basic desktop database application example.

It is uses Flask, Flask-SQLAlchemy, and Flask-Admin to create a basic ‘CRUD’ (create, read, update, and delete) application in a web browser.

It also includes Flask-Migrate, which is helpful for database transformations when you want to make changes to your data models.

The Author and Tag tabs have very basic (mostly default) Admin options. The Book tab demonstrates a greater number of Admin options which you might want to copy.
